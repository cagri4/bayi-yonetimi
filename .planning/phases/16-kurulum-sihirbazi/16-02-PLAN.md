  <name>Task 1: Create wizard dispatcher module</name>
  <files>src/lib/sihirbaz/dispatcher.ts</files>
  <action>
Create src/lib/sihirbaz/dispatcher.ts:

Import:
- `type { Update } from 'grammy/types'`
- `createServiceClient` from `@/lib/supabase/service-client`
- `loadOrCreateSession` from `./session`
- `handleStep` from `./steps`
- `sendTelegramMessage` from `@/lib/telegram/send`

Read BOT_TOKEN from `process.env.TELEGRAM_BOT_TOKEN_SIHIRBAZ || ''`.

Export `async function dispatchSihirbazUpdate(update: Update): Promise<void>`:

1. Extract message from `update.message`. If no message or no `message.text`, return silently (ignore photos, stickers, edits, callback queries).
2. Extract `chatId = message.chat.id` and `text = message.text.trim()`.
3. Create service client: `const supabase = createServiceClient()`.
4. In try/catch block:
   a. Call `loadOrCreateSession(supabase, chatId, text)`.
   b. If session is null:
      - If text starts with `/start`: send Turkish rejection "Bu davet linki artik gecerli degil. Lutfen yetkilinizden yeni bir davet linki isteyin." via sendTelegramMessage.
      - If text does NOT start with `/start`: send "Merhaba! Sisteme kayit olmak icin bir davet linkine ihtiyaciniz var. Lutfen firma yetkilinize basvurun." via sendTelegramMessage.
      - Return.
   c. If session status is 'completed': send "Kurulum zaten tamamlandi. Web panelinize giris yapabilirsiniz." and return (do not call handleStep).
   d. If session status is 'expired': send "Bu kurulum sureci sona erdi. Yeni bir davet linki icin yetkilinize basvurun." and return.
   e. Call `handleStep(supabase, session, chatId, text, BOT_TOKEN)`.
5. Catch block: log error with `console.error('[sihirbaz] dispatch error:', err)` and send "Bir hata olustu. Lutfen tekrar deneyin." to user via sendTelegramMessage.

IMPORTANT: The dispatcher handles terminal state checks (completed, expired) BEFORE delegating to handleStep. This prevents handleStep from needing to check every time.
IMPORTANT: Non-/start messages from unknown users get a helpful message explaining they need an invite link.
