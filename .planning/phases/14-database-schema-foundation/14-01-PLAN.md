For tables with `updated_at` (subscriptions, onboarding_sessions), include a trigger using:
```sql
CREATE OR REPLACE FUNCTION update_updated_at_column()
RETURNS TRIGGER AS $$
BEGIN
  NEW.updated_at = NOW();
  RETURN NEW;
END;
$$ LANGUAGE plpgsql;
```
This function likely already exists from earlier migrations -- use `CREATE OR REPLACE` to be safe. Create the trigger on each table:
```sql
CREATE TRIGGER set_updated_at BEFORE UPDATE ON {table_name}
FOR EACH ROW EXECUTE FUNCTION update_updated_at_column();
```
