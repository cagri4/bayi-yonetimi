   - Description (Turkish): `'Sik sorulan sorulari (SSS) arar ve ilgili sorulari ve cevaplari dondurur.'`
   - input_schema: `{ query: string (required) }` — "Aranacak konu veya soru"

**Handler factory (exported):**

`createEgitimciHandlers(supabase: SupabaseClient<Database>): Map<string, HandlerFn>`
- Returns a Map with exactly 2 entries: `'get_product_info'` and `'get_faq'`
- Handler names MUST match tool names exactly (sync with TOOL_REGISTRY)

**get_product_info handler:**
- Validate query is non-empty string, return `'[Hata: Urun sorgusu bos olamaz]'` if empty
- Query `products` table: `select('id, code, name, description, base_price, stock_quantity, low_stock_threshold')` with `.eq('company_id', context.companyId)`, `.eq('is_active', true)`, `.or('name.ilike.%${query}%,code.ilike.%${query}%')`, `.limit(5)`
- On error: return `'[Hata: ${error.message}]'`
- If no products: return `'[Urun bulunamadi]'`
- Fetch dealer-specific prices: query `dealer_prices` with `.eq('dealer_id', context.dealerId)` and `.in('product_id', products.map(p => p.id))`
- Build priceMap: `new Map((dealerPrices ?? []).map(dp => [dp.product_id, dp.custom_price]))`
- Fetch dealer group discount: query `dealers` with `.select('dealer_group:dealer_groups(discount_percent)')` and `.eq('id', context.dealerId).single()`
- Calculate dealer price: `customPrice ?? base_price * (1 - discountPercent/100)`, rounded to 2 decimals
- Return `JSON.stringify(result)` with fields: code, name, description, base_price, dealer_price, stock, low_stock (boolean)

**CRITICAL: SQL injection prevention for ilike queries:**
Sanitize the query parameter by removing `%` and `_` characters before using in `.or()`. Use: `const safeQuery = query.replace(/[%_]/g, '')`.
