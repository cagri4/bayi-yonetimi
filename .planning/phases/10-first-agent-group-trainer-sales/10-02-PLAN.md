**create_order handler â€” step by step:**

1. **Validate items array:**
   - Extract `items` from input, cast to `Array<{ product_code: string, quantity: number }>`
   - If empty or not an array: return `'[Hata: Siparis kalemleri bos olamaz]'`
   - If items.length > 20: return `'[Hata: Tek sipariste en fazla 20 urun olabilir]'`

2. **Resolve products by code:**
   - Extract unique product codes: `[...new Set(items.map(i => i.product_code))]`
   - Query `products`: `select('id, code, name, base_price, stock_quantity')` with `.eq('company_id', context.companyId)`, `.eq('is_active', true)`, `.in('code', productCodes)`
   - If no products found: return `'[Hata: Belirtilen urunler bulunamadi]'`
   - Check for missing products: compare found codes vs requested codes. If any missing, return `'[Hata: Su urunler bulunamadi: ${missingCodes.join(', ')}]'`

3. **Stock validation (Pitfall 5 â€” inline stock check):**
   - For each requested item, check `product.stock_quantity >= item.quantity`
   - If any insufficient: return `'[Stok yetersiz: ${product.name} icin sadece ${product.stock_quantity} adet mevcut, ${item.quantity} adet istendi]'`

4. **Get dealer pricing:**
   - Query `dealer_prices` with `.eq('dealer_id', context.dealerId)`, `.in('product_id', products.map(p => p.id))`
   - Build priceMap
   - Query `dealers` for `dealer_groups(discount_percent, min_order_amount)` with `.eq('id', context.dealerId).single()`
   - Extract discountPercent and minOrderAmount

5. **Build order items array:**
   - For each item in input: find product, get dealer price (custom or discounted), calculate total
   - Round all prices to 2 decimals: `Math.round(value * 100) / 100`
   - Calculate subtotal

6. **Minimum order amount check:**
   - If `subtotal < minOrderAmount`: return `'[Minimum siparis tutari ${minOrderAmount} TL. Mevcut tutar: ${subtotal.toFixed(2)} TL]'`

7. **Get pending status ID:**
   - Query `order_statuses`: `.select('id').eq('code', 'pending').single()`
   - If not found: return `'[Hata: Siparis durumu bulunamadi]'`

8. **Generate order number:**
   - Call `supabase.rpc('generate_order_number')`
   - Fallback: `'ORD-${Date.now()}'`

9. **Insert order:**
   - Insert into `orders`: `{ company_id: context.companyId, order_number, dealer_id: context.dealerId, status_id: pendingStatus.id, subtotal, discount_amount: 0, total_amount: subtotal }`
   - Use `.select().single()` to get inserted record
   - On error: return `'[Hata: Siparis olusturulamadi: ${error.message}]'`

10. **Insert order items:**
    - Insert array into `order_items`: each item gets `{ company_id: context.companyId, order_id: order.id, product_id, product_code, product_name, quantity, unit_price, total_price }`
    - On error: rollback by deleting the order `supabase.from('orders').delete().eq('id', order.id)`, return error

11. **Insert status history:**
    - Insert into `order_status_history`: `{ company_id: context.companyId, order_id: order.id, status_id: pendingStatus.id, notes: 'Siparis Telegram uzerinden olusturuldu' }`

12. **Return success:**
    - `JSON.stringify({ success: true, order_number, order_id: order.id, total_amount: subtotal, items_count: orderItems.length })`
