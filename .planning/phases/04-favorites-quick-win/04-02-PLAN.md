  <action>
Create Zustand store following v1 cart.ts pattern:

1. Define FavoritesStore interface:
   - favoriteIds: Set&lt;string&gt; (product IDs)
   - addFavorite(productId: string): void
   - removeFavorite(productId: string): void
   - toggleFavorite(productId: string): void
   - isFavorite(productId: string): boolean
   - hydrate(productIds: string[]): void
   - clear(): void

2. Create useFavoritesStore with persist middleware:
   ```typescript
   export const useFavoritesStore = create&lt;FavoritesStore&gt;()(
     persist(
       (set, get) => ({
         favoriteIds: new Set&lt;string&gt;(),
         // ... implementations
       }),
       {
         name: 'dealer-favorites-storage',
         // Handle Set serialization to/from Array
         partialize: (state) => ({
           favoriteIds: Array.from(state.favoriteIds),
         }),
         onRehydrateStorage: () => (state) => {
           if (state && Array.isArray(state.favoriteIds)) {
             state.favoriteIds = new Set(state.favoriteIds as unknown as string[])
           }
         },
       }
     )
   )
   ```

3. Implement all methods:
   - addFavorite: Create new Set, add ID, update state
   - removeFavorite: Create new Set, delete ID, update state
   - toggleFavorite: Check if exists, call add or remove
   - isFavorite: Return favoriteIds.has(productId)
   - hydrate: Set favoriteIds to new Set from array (for server sync)
   - clear: Reset to empty Set

Note: Use Set for O(1) lookup performance. Handle serialization carefully since JSON.stringify doesn't support Set.
