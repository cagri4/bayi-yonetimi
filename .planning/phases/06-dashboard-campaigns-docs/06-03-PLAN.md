1. **Create src/lib/actions/announcements.ts**:

   Dealer queries:
   - `getAnnouncements()`:
     - SELECT announcements WHERE is_active = true
     - AND published_at <= NOW()
     - AND (expires_at IS NULL OR expires_at >= NOW())
     - LEFT JOIN announcement_reads for current dealer to get read status
     - ORDER BY priority DESC, published_at DESC
     - Returns array with { ...announcement, is_read: boolean }

   - `getUnreadAnnouncementCount()`:
     - Count unread announcements for badge display
     - Returns number

   Dealer mutations (with 'use server'):
   - `markAnnouncementAsRead(announcementId: string)`:
     - Get current dealer from user
     - INSERT INTO announcement_reads (announcement_id, dealer_id)
     - ON CONFLICT DO NOTHING (already read)
     - Revalidate /announcements path
     - Follow pattern from favorites.ts toggle

   - `markAllAnnouncementsAsRead()`:
     - Bulk insert read receipts for all unread
     - Use INSERT ... SELECT pattern

   Admin mutations:
   - `createAnnouncement(formData)`: Insert announcement
   - `updateAnnouncement(id, formData)`: Update
   - `deleteAnnouncement(id)`: Soft delete or hard delete

2. **Create src/components/announcements/announcement-card.tsx**:
