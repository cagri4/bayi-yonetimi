Create the TokenBudget class at `src/lib/agents/token-budget.ts` that enforces per-dealer daily token limits.

**Imports:**
- `createServiceClient` from `@/lib/supabase/service-client`
- `SOFT_TOKEN_LIMIT, HARD_TOKEN_LIMIT` from `./types`
- `TokenUsageRecord` from `./types`

**Class: TokenBudget**

Method 1: `async checkBudget(dealerId: string): Promise<{ allowed: boolean; remaining: number; reason?: string }>`
- Get today's date as 'YYYY-MM-DD' string: `new Date().toISOString().slice(0, 10)`
- Query daily_token_usage for the dealer + today
- If tokens_used >= HARD_TOKEN_LIMIT: return `{ allowed: false, remaining: 0, reason: 'Gunluk token limitinize ulastiniz (100.000 token). Yarin tekrar deneyin.' }`
- If tokens_used >= SOFT_TOKEN_LIMIT: log warning via console.warn, return allowed: true with remaining count
- Otherwise: return allowed: true with remaining = HARD_TOKEN_LIMIT - used

Method 2: `async recordUsage(dealerId: string, usage: TokenUsageRecord): Promise<void>`
- Calculate totalTokens = inputTokens + outputTokens + cacheReadTokens + cacheWriteTokens
- Call `supabase.rpc('increment_daily_token_usage', { p_dealer_id: dealerId, p_date: today, p_tokens: totalTokens })`
- Log the token usage for observability: `console.log(\`[token-budget] dealer=${dealerId} tokens=${totalTokens} (in=${usage.inputTokens} out=${usage.outputTokens} cache_r=${usage.cacheReadTokens} cache_w=${usage.cacheWriteTokens})\`)`

Method 3: `async getUsage(dealerId: string): Promise<number>`
- Simple query to get today's tokens_used for the dealer
- Returns 0 if no row exists

**Important:** The class creates a service client once in constructor (or lazily). Do NOT create a new client per method call.

The Turkish error message in checkBudget is intentional — all agent responses to dealers are in Turkish per project convention.
